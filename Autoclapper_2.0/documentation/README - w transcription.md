# AutoClapper 2.0 - Script & Clapper Generator

A web-based tool for generating script documents and clapper images for video production.

## Project Structure

```
autoclapper/
├── app.py                      # Python Flask backend server
├── autoclapper-interface.html  # Web interface (open in browser)
├── requirements.txt            # Python dependencies
├── templates/
│   ├── script_template.docx    # Your script template (REQUIRED)
│   └── clapper_blank.png       # Your blank clapper image (REQUIRED)
└── README.md                   # This file
```

## Setup Instructions

### 1. Install Python Dependencies

Open Command Prompt and navigate to the project folder, then run:

```bash
pip install -r requirements.txt
```

**Note**: The batch file (see step 3) will do this automatically if needed.

### 2. Add Your Template Files

Place your template files in the `templates/` folder:
- `script_template.docx` - Your Word document template with the 4-table structure
- `clapper_blank.png` - Your blank clapper image (1920x1080)

### 3. Start AutoClapper (Easy Way!)

Simply double-click `start_autoclapper.bat`

This batch file will:
- Check if Python is installed
- Install dependencies if needed
- Create the templates folder if missing
- Start the server
- Automatically open the web interface in your browser

**Keep the Command Prompt window open while using AutoClapper!**

### Alternative: Manual Start

If you prefer to start manually:

1. Open Command Prompt and run:
```bash
python app.py
```

2. Open `autoclapper-interface.html` in your browser OR navigate to `http://localhost:5000`

## Usage

1. **Select Output Directory**: Click "Browse..." and enter the path where you want to save files
2. **Fill in the form fields**:
   - General Information (shared between script and clapper)
   - Script Details (script-only fields)
   - Clapper Details (clapper-only fields)
   - Script Content (Video/Vision and Audio text)
3. **Optional: Use Audio Transcription**:
   - Click "Attempt Auto Transcription" button in the Audio section
   - Select an audio or video file from your ad
   - Wait 30-60 seconds for transcription to complete
   - Edit the auto-generated text as needed
4. **Format your text** using the toolbar buttons (Bold, Italic, Underline, Highlight)
5. **Click "Create and Save Documents"**
6. Check the status area for confirmation of successful generation

## Audio Transcription Feature (Optional)

AutoClapper includes optional audio transcription powered by OpenAI's Whisper. This feature can automatically transcribe spoken audio from your ad videos.

**To enable audio transcription:**
```bash
pip install openai-whisper
```

**Note:** The first time you use transcription, Whisper will download a ~140MB model file. Subsequent uses will be faster.

**Supported formats:** MP4, MP3, WAV, MOV, AVI, and most common audio/video formats

**How to use:**
1. Click the "Attempt Auto Transcription" button in the Audio section
2. Select your ad's audio or video file
3. Wait for transcription (typically 30-60 seconds)
4. Review and edit the transcribed text

If Whisper is not installed, the transcription button will still appear but will show an error message when clicked, with instructions on how to install it.

## Auto-Filled Fields

These fields are automatically filled and cannot be edited:
- **Date** (in script): Current date in DD/MM/YY format
- **Format** (in clapper): Always "16:9"

## Output Files

Generated files are saved with the naming format:
- `Script_[KeyNo]_[Client].docx`
- `Clapper_[KeyNo]_[Client].png`

## Template Structure Reference

### Script Template (script_template.docx)

**Table 1**: Header graphics (unchanged)

**Table 2**: Job details (5 rows × 4 columns)
- Column 2: Client, Product, Duration, Key No, Notes
- Column 4: Contact, Ph No, Date, On Air, Prod $

**Table 3**: Script content (2 rows × 2 columns)
- Row 2, Col 1: Video/Vision
- Row 2, Col 2: Audio

**Table 4**: Generator signature (1 row × 1 column)
- Auto-generated: "Generated by AutoClapper 2.0 on [date]"

### Clapper Template (clapper_blank.png)

- Resolution: 1920 × 1080
- Format: PNG
- Fields: Key No, Client, Product, Duration, Agency, Format (16:9)

## Integration Notes

The clapper generation function is ready to integrate your existing Python script.
When you have access to your clapper script, it will be added to the `generate_clapper_image()` function in `app.py`.

## Troubleshooting

**"Template not found" error:**
- Ensure `script_template.docx` and `clapper_blank.png` are in the `templates/` folder
- Check that filenames match exactly (case-sensitive)

**"Could not connect to backend" error:**
- Make sure `python app.py` is running
- Check that the terminal shows "Running on http://localhost:5000"

**Generated files not appearing:**
- Verify the output directory path is correct
- Check you have write permissions for the output directory
